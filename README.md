## 🚀 نظرة عامة (Overview)
نظام **WanoHost** هو منصة ويب متطورة تعتمد على معمارية الوحدات (Modular Architecture). تم بناء هذا النظام بالكامل باستخدام **JavaScript** الصافي (Vanilla JS) لضمان أقصى سرعة استجابة وأداء، مع تصميم مستقبلي يعتمد على تقنيات الـ **Glassmorphism**.

تم تصميم هذا المشروع ليكون متوافقاً بالكامل مع استضافة **GitHub Pages**، حيث يدير أكثر من 30 ملفاً برمجياً مترابطاً بشكل ديناميكي.

---

## 🛠 الهيكلية البرمجية (Project Structure)
يحتوي النظام على تقسيم احترافي للمجلدات لضمان سهولة التطوير والصيانة:

* **`/js/core/`**: يحتوي على المحركات الأساسية (Config, Data Manager, Event Bus).
* **`/js/modules/`**: يضم الوحدات الوظيفية (Auth, Security, UI Engine, Backup, Terminal).
* **`/css/`**: ملفات التصميم المقسمة (Global, Animations, Dashboard, Modals).
* **الواجهات**: (`index.html`, `dashboard.html`, `404.html`).

---

## ✨ المميزات الرئيسية (Key Features)

* **Modular Design**: نظام مقسم إلى 30 ملف لسهولة الإدارة.
* **Security Layer**: نظام تشفير محلي للبيانات وفحص التهديدات.
* **Dynamic Dashboard**: لوحة تحكم ذكية تعرض حالة الخوادم والبيانات فورياً.
* **Future UI**: واجهة مستخدم متطورة تدعم الأنيميشن المعقد (Glitch Effects).
* **Mobile Optimized**: تهيئة كاملة للعمل على الهواتف الذكية مع دعم لمسات الـ Touch.
* **Integrated Terminal**: محاكي أوامر داخلي لعرض عمليات النظام في الوقت الفعلي.

## 🚀 التشغيل والاستضافة (Deployment)

النظام جاهز للعمل مباشرة على **GitHub Pages**. للتشغيل:
1. ارفع كافة الملفات والمجلدات إلى مستودعك.
2. توجه إلى إعدادات المستودع (**Settings**).
3. اختر **Pages** وقم بتفعيل الـ Branch الرئيسي (**main**).
4. سيتم توليد الرابط الرسمي لنظامك خلال ثوانٍ.
