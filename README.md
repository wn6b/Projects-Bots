## 🌟 نبذة عن المشروع
**Projects Bots** هي منصة برمجية متطورة مصممة لتكون الوجهة الأولى لإدارة ونشر مشاريع البوتات والتطبيقات. تم بناء المنصة لتعمل بالكامل بتقنيات الـ AI، حيث توفر تجربة مستخدم فريدة تجمع بين السرعة، الأمان، والتصميم المستقبلي الذي يليق بمعايير عام 2026.

المشروع ليس مجرد موقع رفع ملفات، بل هو "نظام إيكولوجي" متكامل يحتوي على لوحة تحكم للمالك (Owner) وأخرى للمستخدمين، مع ربط مباشر بنماذج الذكاء الاصطناعي (Claude AI) لفحص الملفات ومراجعة الحسابات.

---

## 🚀 المميزات الرئيسية (Core Features)

### 🤖 1. نظام AI Scanner الأمني
* **فحص ذكي:** فحص تلقائي لكل مشروع يتم رفعه للتأكد من خلوه من الفيروسات أو الكود المشبوه.
* **تحليل Claude AI:** الربط مع موديل `claude-sonnet-4-20250514` لتحليل وصف المشروع ومحتواه برمجياً.
* **رفض تلقائي:** منع نشر أي محتوى ضار فوراً بواسطة الـ AI.

### 👑 2. لوحة تحكم المالك (Owner Dashboard)
* **إحصائيات حية:** مراقبة عدد المشاريع، المطورين، التحميلات، والمستخدمين بشكل فوري.
* **إدارة المشاريع:** إمكانية إضافة، حذف، وتعديل المشاريع بسهولة.
* **سجل النشاط (Activity Log):** تتبع كل حركة تحدث داخل المنصة (تسجيل دخول، رفع، حذف).
* **إدارة المطورين:** بوابة خاصة لمتابعة المطورين المنضمين للمنصة.

### 👤 3. لوحة تحكم المستخدم (User Dashboard)
* **تصفح ذكي:** واجهة بسيطة لاستعراض المشاريع المتاحة.
* **تحميل مباشر:** إمكانية تحميل الملفات المرفقة للمشاريع بضغطة زر.
* **بوابة المطورين:** قسم مخصص لتوجيه المستخدمين لكيفية الانضمام كفريق عمل.

### 🎨 4. واجهة مستخدم (UI/UX) مستقبلية
* **تصميم 2026:** استخدام تدرجات لونية (Gradients)، تأثيرات زجاجية (Glassmorphism)، وجزيئات متحركة (Particles).
* **تجربة سلسة:** شاشة تحميل (Loader) تفاعلية ونظام تنقل ذكي بين الأقسام.

## 🛠 التقنيات المستخدمة (Tech Stack)
* **Frontend:** HTML5, CSS3 (Custom Variables & Animations).
* **Logic:** Pure JavaScript (ES6+) - **No Frameworks needed.**
* **AI Integration:** Anthropic API (Claude AI).
* **Database:** LocalStorage API (لضمان السرعة والعمل بدون سيرفر خارجي في المرحلة الحالية).
* **Fonts:** Syne & Space Mono (للطابع التقني) + Tajawal (للعربية الاحترافية).

## 📦 طريقة التركيب والتشغيل (Setup Instructions)

### 💻 التشغيل المحلي (Localhost)
1. قم بتحميل ملفات المشروع (`index.html`, `style.css`, `script.js`).
2. ضع جميع الملفات في مجلد واحد.
3. افتح ملف `index.html` في أي متصفح حديث (يفضل Chrome أو Edge).

### 🌐 الاستضافة على سيرفر (Normal Host)
1. ارفع الملفات إلى مدير الملفات (File Manager) في استضافتك (cPanel أو غيرها).
2. تأكد من وضع الملفات في مجلد `public_html`.
3. لا يحتاج المشروع لقاعدة بيانات SQL لأنه يعتمد على `LocalStorage` و `Anthropic API`.

### 📱 التشغيل عبر Termux (Termux Host)
إذا كنت تستخدم هاتفك كخادم:
