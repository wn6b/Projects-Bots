## 1. مقدمة المشروع (Introduction)
مرحباً بك في **Projects Bots**، المنصة الأذكى والأكثر تطوراً لعام 2026، والمصممة خصيصاً لاستضافة ومشاركة البوتات، المواقع، والأدوات البرمجية. تعتمد هذه المنصة على دمج تقنيات الـ Front-end الحديثة مع محركات الذكاء الاصطناعي (Claude AI) لتقديم تجربة مستخدم لا مثيل لها، خالية من التدخل البشري اليدوي في عمليات الفحص والمراجعة.

هذا المشروع ليس مجرد واجهة رسومية، بل هو نظام تشغيل مصغر (Web-based OS) يدير المستخدمين، المطورين، والملفات بشكل آمن وسلس عبر بيئة (No-Database Architecture) تعتمد على الـ Storage المتزامن.

## 2. الرؤية والهدف (Vision & Mission)
في عالم يمتلئ بالبرمجيات الخبيثة والملفات غير الآمنة، جاءت فكرة **Projects Bots** لتوفير "ملاذ آمن" للمستخدمين. 
* **للمستخدم العادي:** بيئة آمنة 100% لتحميل البوتات والإضافات دون خوف، حيث أن كل ملف مرفوع قد اجتاز فحصاً عميقاً من قبل الـ AI.
* **للمطور:** منصة احترافية (Projects Developers) تليق بجهده، توفر له أدوات رفع متقدمة، إحصائيات دقيقة، ووصول عالمي لآلاف المستخدمين.

## 3. البنية التحتية (System Architecture)
تم بناء المشروع ليكون قادراً على العمل مباشرة على **GitHub Pages** (Static Hosting) دون الحاجة إلى خوادم (Backend Servers) تقليدية كـ Node.js أو Python.
* **التخزين الذكي (Smart Local Storage):** يتم حفظ بيانات المستخدمين، المشاريع، الجلسات، والإحصائيات عبر بيئة الـ LocalStorage المشتركة بين صفحات الموقع.
* **Vector-Based UI:** تم التخلي تماماً عن الرموز التعبيرية (Emojis) العادية، واستبدالها بنظام أيقونات مبني على الفيكتور (FontAwesome) لضمان دقة العرض وتناسق المظهر التقني (Geometric Brutalism) الخاص بعام 2026.
* **Real-time DOM Manipulation:** تفاعل فوري مع واجهة المستخدم بدون إعادة تحميل الصفحة (Single Page Application logic).

## 4. النظام المزدوج (The Dual-Portal System)
المشروع ينقسم إلى عالمين متصلين ببعضهما البعض:

### أ. العالم الأول: Projects Bots (الموقع الأساسي)
وهو الواجهة الموجهة للمستخدمين العاديين ومالك النظام (Owner). يحتوي على:
* **واجهة دخول ذكية:** تتكيف مع نوع الحساب (مستخدم / مطور / Owner).
* **لوحة تحكم الأونر (Owner Dashboard):** السيطرة المطلقة على النظام، حذف المشاريع، مراقبة النشاطات، مسح البيانات، وإدارة المطورين.
* **لوحة المستخدم العادي (User Dashboard):** استعراض المشاريع وتحميلها بأمان بضغطة زر (Real File Download).

### ب. العالم الثاني: Projects Developers (بوابة المطورين - مسار `pd/`)
محطة العبور الصارمة لأي مبرمج يرغب في الانضمام. تحتوي على:
* **Multi-step Registration Form:** نظام تسجيل معقد يطلب تفاصيل دقيقة عن المبرمج ومشروعه.
* **AI Evaluation:** لا يتدخل الأونر في قبول المطورين! يقوم الـ AI بتحليل منطقية الطلب وقبوله أو رفضه فورياً.
* **Sync Protocol:** فور قبول المطور، يتم إرسال بياناته برمجياً إلى الموقع الأساسي ليتمكن من تسجيل الدخول.

## 5. الذكاء الاصطناعي والأمن (AI & Security Core)
النظام محمي بطبقات أمنية متطورة تمثل الجيل الجديد من حماية الويب:
1. **Math CAPTCHA:** نظام تحقق رياضي ديناميكي يمنع هجمات الـ Spam.
2. **Cloudflare Turnstile Simulation:** نظام تحقق أمني لمنع الروبوتات من تجاوز التسجيل.
3. **AI Scanner (Deep Scan):** * أي ملف يتم رفعه (ZIP, RAR, JS, HTML) يتم تمرير بياناته ووصفه إلى نموذج `claude-sonnet-4-20250514`.
   * يقوم الـ AI بتحليل المنطق البرمجي (Logic Validation) والبحث عن أي دلالات برمجية خبيثة (Malware/Backdoors).
   * يتم توليد تقرير آلي من الـ AI يُعرض في الـ Terminal الخاص بالنظام.
   * **لا يتم نشر أي مشروع دون اجتياز الفحص بنجاح.**

## 6. دليل المطورين (Developers Guide)
إذا كنت مطوراً وترغب في رفع أعمالك، اتبع المسار التالي:
1. انتقل إلى بوابة المطورين (`/pd/index.html`).
2. انقر على "انضم كمطور".
3. قم بتعبئة بياناتك الشخصية وبيانات مشروعك بدقة.
4. اجتز اختبارات الأمان (CAPTCHA).
5. انتظر بضع ثوانٍ حتى يقوم محرك الـ AI بمراجعة طلبك.
6. في حال القبول (Approved)، سيتم توجيهك للموقع الأساسي.
7. قم بتسجيل الدخول بحسابك الجديد، وستظهر لك واجهة المطور لرفع ملفاتك الفعلية.

    ├── style.css       # تصميم بوابة المطورين
    └── script.js       # ربط الـ AI و الـ API لتقييم المطورين
